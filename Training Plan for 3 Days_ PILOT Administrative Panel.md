### **Training Plan for 3 Days: PILOT Administrative Panel**

Goal: To train an employee in the basics of working with the admin panel: creating accounts, creating partners, configuring modules and configurations, and solving typical user tasks.

---

### **Day 1: Interface Familiarization and Basic Operations**

#### **📘 Theory:**

* Introduction to the Administrative Panel  
* Interface Structure: Graphical Panel, Tabs, Workspace  
* Personal User Settings  
* The "Contracts" Section: Creation, Editing, Basic Settings

#### **🛠 Practice:**

1. Log into the Administrative Panel.  
2. Change personal settings: change name, password, add a photo.  
3. Create a new contract:  
   * Fill in the basic settings: name, email, phone, organization type, account type, tariff.  
   * Activate modules (e.g., Video, Drivers).  
   * Save the contract.

#### **✅ Test Assignment:**

* Answer the questions:  
  1. What is a "stock account" and what is it used for?  
  2. What types of accounts are there and what is the difference between them?  
  3. How do you add a configuration to a contract?  
* Create a contract with the "Prepayment" account type and activate the "Drivers" module.

---

### 

### **Day 2: Working with Objects, Partners, and Finances**

#### **📘 Theory:**

* The "Vehicles" Section: Configuration, Transfer Between Contracts  
* Object Configurations: Configuring Object Parameters, Blocking, Setting Tariffs.  
* The "Partners" Section: Creation, Configuration, Settings  
* The "Finances" Section: Payments, Debits, Tariffs

#### **🛠 Practice:**

1. Transfer an object (vehicle) to the created contract.  
2. Configure configurations for the object:  
   * Add a tariff.  
   * Configure blocking with a block date.  
   * Add a configuration in the object settings for speed control.  
3. Create a partner:  
   * Fill in the basic data, currency, tariff.  
   * Configure SMTP for email functionality.  
4. Add a payment in the "Finances" section.

#### **✅ Test Assignment:**

* Answer the questions:  
  1. How do you transfer an object from one contract to another?  
  2. Which configurations are used for speed control?  
  3. How do you add a manual subscription fee debit?  
* Create a partner and transfer a contract to them. Configure the "Address in Online Tree" configuration.

---

### **Day 3: Modules, Notifications, Security, and Rebranding**

#### **📘 Theory:**

* Overview of Modules: Video, Drivers, Geofences, Notifications, etc.  
* Configuring Notifications: Email Templates, Payment Reminders  
* Security: Two-Factor Authentication (TOTP, Email)  
* Rebranding: Configuring Logos, Colors, Links

#### **🛠 Practice:**

1. Activate the "Geofences" module for a contract.  
2. Configure an email template for low balance notifications.  
3. Enable two-factor authentication for a partner (TOTP or Email).  
4. Configure rebranding: upload a logo, choose a theme, check the display on the login page.

#### **✅ Test Assignment:**

* Answer the questions:  
  1. How do you activate the "Analytics" module for a contract?  
  2. How do you set up TOTP for system login?  
  3. What settings can be changed in rebranding?  
* Configure and test the "Notifications" module. Add the "Low Balance Emails" configuration.

---

### **Final Test (After 3 Days):**

1. Create a contract with prepayment, add an object, and configure the configuration for mileage by CAN.  
2. Create a partner, assign a contract to them, and configure SMTP.  
3. Activate the "Video" module and configure streaming in a new tab.  
4. Write instructions for the user: "How to change the password in the personal account".
